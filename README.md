# 👁️ Eye-Controlled Cursor

An AI-powered application that allows users to control their mouse cursor using eye movements and facial landmarks. This project leverages Computer Vision to create a seamless, hands-free human-computer interaction experience.

---

## 🚀 Key Features
* **Real-time Tracking:** High-accuracy iris and facial landmark detection.
* **Hands-Free Interaction:** Control cursor movement and clicking without a physical mouse.
* **Ready to Use:** Standalone executable available for Windows users.
* **Efficient Architecture:** Modular code design for low-latency performance.
